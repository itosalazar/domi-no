DOMI-NO

A clean, modern take on classic dominoes.

DOMI-NO reimagines the traditional domino game through a minimal, Bauhaus-inspired interface focused on clarity, rhythm, and satisfying play. The experience strips away unnecessary visual noise while keeping the game intuitive, responsive, and visually distinctive.

About

DOMI-NO is a functional, browser-based domino game designed around a simple idea: classic gameplay deserves a sharper interface.

The visual system combines geometric composition, strong typography, restrained layouts, and bold accent colors to create an experience that feels modern without losing the familiarity of the original game.

Features

* Fully playable domino game
* Clean, responsive interface
* Bauhaus-inspired visual direction
* Minimal and intuitive game controls
* Clear turn and game-state feedback
* Responsive layout for different screen sizes
* Fast browser-based gameplay
* No installation required

Design Direction

The interface follows a minimal and functional design philosophy:

* Strong geometric structure
* Clear visual hierarchy
* Purposeful use of color
* Generous negative space
* Simple, direct interactions
* Reduced visual clutter
* Emphasis on the domino pieces and gameplay

The goal is not decoration for decoration’s sake. Every visual element should support readability, interaction, or the rhythm of play.

Local Development

Clone the repository:

git clone YOUR_REPOSITORY_URL

Navigate into the project folder:

cd YOUR_PROJECT_FOLDER

Install dependencies:

npm install

Start the local development server:

npm run dev

Open the local URL shown in Terminal, typically:

http://localhost:5173

Production Build

Create a production build:

npm run build

Preview the production build locally:

npm run preview

Deployment

DOMI-NO is prepared for deployment on Vercel.

Typical deployment flow:

1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Confirm the framework and build settings.
4. Deploy.
5. Use the generated .vercel.app URL to access and share the game.
